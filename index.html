PROJECTS

A list of different code projects in which I have worked on or have been involved in.

Generative Design System

A prototype of generative design system using neuroevolution. This system allowed users to optimize 2D and 3D truss structures for multi-objective optimization problems. Neural network agents are evolved to optimize design solutions allowing for re-deployment to other similar design problems. I built a 3D modeling environment using Flask, React and React Three Fiber allowing users to generate truss structures and boundary conditions. After training, users could select design solutions from a Pareto chart and instantly view high resolution 3D models of them.

Repo
Stiffened Plate Generative Design

This project is a generative design methodology for stiffened plates. Using neuroevolution, agents are evolved to respond to simulation states with the goal of minimizing the total volume and displacement of the initial design. The agents' action spaces control the width of stiffeners reinforcing the plate. The state space includes the displacement and volume of the stiffeners and the displacement of the plate sections. The state features are found using the Finite Element Analysis (FEA) solver Calculix, where the stiffeners are represented as beam elements and the plate sections as shell elements.

Signed Distance Function (SDF) Experiments

A website of different experiments using signed distance functions (SDFs). The website uses three.js and Open GL Shader Language (GLSL) for rendering SDF models using ray marching.
